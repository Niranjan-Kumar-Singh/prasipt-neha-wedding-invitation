# Prasipt & Neha - Luxury Wedding Invitation

A premium, interactive, and beautifully designed digital wedding invitation for Prasipt Kumar Singh and Neha Kumari.

## Features
- **Luxury Aesthetic**: High-end typography (Cormorant Garamond, Alex Brush) and a royal maroon/gold color palette.
- **Interactive Experience**: Interactive envelope opening animation, subtle floating orbs, and scrolling reveals.
- **Background Music**: Integrated audio player for ambient romantic music.
- **Responsive Design**: Looks stunning on both desktop and mobile devices.

## Tech Stack
- HTML5
- CSS3 (Tailwind CSS via CDN for utility classes)
- Vanilla JavaScript
- Google Fonts
